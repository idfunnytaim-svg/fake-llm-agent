# Fake LLM Agent

## 📌 Project Overview
이 프로젝트는 **LLM Agent 구조를 이해하기 위한 학습용 예제 프로젝트**입니다.  
실제 대규모 LLM 연동보다는,  
- Agent의 역할 분리
- Tool 호출 흐름
- Workflow 구조  
를 코드 레벨에서 파악하는 것을 목표로 합니다.

---

## 🛠️ Development Environment

- **Language**: Python
- **IDE**: PyCharm
- **Version Control**: Git / GitHub
- **OS**: macOS
